# Ansible Collection - enricocirignaco.raspberry

Dedicated Ansible collection for Raspberry Pi specific provisioning and host configuration.

## Scope

This collection is reserved for Raspberry Pi focused automation and role development.

## Included roles

No roles have been published yet.

## Requirements

- `ansible-core >= 2.14`

## Build

```bash
ansible-galaxy collection build ansible_collections/enricocirignaco/raspberry
```

## Install locally

```bash
ansible-galaxy collection install ./enricocirignaco-raspberry-1.0.0.tar.gz
```
